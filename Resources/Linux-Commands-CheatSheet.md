# Linux Commands for Blue Team

## Navigation
| Command | Description | Example |
|---------|-------------|---------|
| pwd | Print working directory | `pwd` |
| ls | List files | `ls -la` |
| cd | Change directory | `cd /var/log` |

## File Operations
| Command | Description | Example |
|---------|-------------|---------|
| cat | View file | `cat file.txt` |
| grep | Search in file | `grep "error" log.txt` |
| tail | View last lines | `tail -f /var/log/syslog` |
